qStopMotion installation CD

Please start

qstopmotion-1.0.1-win32.exe

to install qStopMotion on your computer.
The installation program starts the installer for the gstreamer
automatically if necessary.

This software use the following open source projects.
* Qt (http://qt.digia.com/)
* gstreamer (http://gstreamer.freedesktop.org/)
* FFmpeg (http://ffmpeg.org/)
